Day-28 Shopping Card UseContextAPI
This project is a shopping cart application built with React.js that demonstrates the use of the Context API for state management. 
The application allows users to add, remove, and manage items in their shopping cart.

Features
Add/Remove Items: Users can add items to the cart and remove them as needed.
Cart Management: View the current items and total price in the cart.
State Management: Utilizes React's Context API for managing the state of the shopping cart across the application.
Technologies Used
React.js: Frontend framework for building the user interface.
Context API: React's built-in feature for managing state.
